Developed in the Eclipse IDE environment and written in Java
The src folder holds the Java files.
The iofiles folder holds the JSON file and output file.

Takes a JSON file of a Shakespear play, in this instance is MidsummerNightsDream, and writes it to a text file.

- PlayWriter.java class takes a Json file and writes it to a text file.
- PlayJSONReader.java Reads in the JSON file and stores it in a buffer.
- AccessorUtils.java Helps parse the JSON file to allow proper formatting when printing.
